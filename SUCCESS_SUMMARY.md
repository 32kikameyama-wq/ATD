## 📁 プロジェクト構造

```
ATDシステム/
├── app.py                 # メインアプリ
├── config.py              # 設定
├── models.py              # データベースモデル
├── routes.py              # メインルート
├── auth.py                # 認証
├── tasks.py               # タスク管理
├── templates/             # HTML テンプレート
│   ├── auth/              # 認証ページ
│   └── tasks/             # タスクページ

├── requirements.txt       # 依存パッケージ
├── Procfile              # デプロイ用
└── render.yaml            # Render 設定
```

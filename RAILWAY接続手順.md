# Railway.app リポジトリ接続手順

## 🎯 現在の状況

Railway.app にはアクセスできていますが、まだリポジトリ（GitHub 上の ATD）が接続されていません。

## ✅ 次のステップ

### Step 1: 新しいプロジェクトを作成

1. **画面左上の「New」ボタンをクリック**
   - または、「Deploy a new project」ボタンをクリック

2. **「Deploy from GitHub repo」を選択**

### Step 2: GitHub でログイン

1. **GitHub アカウントでログイン**
   - まだログインしていない場合
   - GitHub の認証ページが表示される

2. **アクセス許可を付与**
   - Railway がリポジトリにアクセスできるようにする

### Step 3: リポジトリを選択

1. **リポジトリ一覧から選択**
   - あなたの GitHub アカウントのリポジトリが表示される
   - **「ATD」または「32kikameyama-wq/ATD」を探す**

2. **「Deploy」ボタンをクリック**

### Step 4: PostgreSQL を追加

1. **プロジェクトページが開く**
   - 「Deploying...」と表示される
   - 数分で完了

2. **「+ New」ボタンをクリック**
   - 左側のメニューまたは画面下部にボタンがある

3. **「Database」を選択**

4. **「Add PostgreSQL」をクリック**
   - 自動的にデータベースが作成され、接続される

### Step 5: 完了！

**これで完了！**
- URL が発行される
- データは永続保存される

## 📸 画面案内

現在表示されているのはダッシュボードです：
- 「Deploy a new project」ボタンをクリック
- または左上の「New」ボタンをクリック

## ⚠️ わからない場合

**「New」または「Deploy a new project」をクリックしてください**

そこから GitHub リポジトリを選択できます。
